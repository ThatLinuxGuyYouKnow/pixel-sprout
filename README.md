## Story

You are a newly sprouted sapling in a world consumed by darkness. The ancient **Golden Seed**, the source of all life and light, has been stolen and hidden deep within the earth.

Your mission is to descend into the perilous depths, survive against the creatures lurking in the shadows, and retrieve the Lost Seed to restore nature to the surface world.

## Controls

- **Arrow Keys / WASD**: Move
- **Space / Enter**: Interact / Attack / Talk
- **.** (Period): Wait a turn (Rest and Heal)

## Features

- **Procedural Dungeons**: Every run is unique with different layouts.
- **Quest System**: Help the lost spirits of the dungeon to gain aid.
- **Turn-Based Combat**: Strategic movement and positioning are key.
- **Retro Pixel Art**: A nostalgic visual style with CRT effects.

## Run Locally

**Prerequisites:** Node.js

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key (optional for basic play, required for AI narration).
3.  Run the app:
    ```bash
    npm run dev
    ```
